# GuardaBotsques

Proyecto de robótica y automatización del IES Alonso Quijano.

## Descripción

GuardaBotsques es un proyecto educativo desarrollado en el IES Alonso Quijano que combina principios de robótica, programación y resolución de problemas prácticos. El proyecto tiene como objetivo fomentar el aprendizaje STEM (Ciencia, Tecnología, Ingeniería y Matemáticas) mediante la construcción y programación de robots autónomos o semiautónomos.

## Características

- Diseño modular de robots
- Programación en lenguajes accesibles para estudiantes
- Sensores y actuadores para interacción con el entorno
- Documentación paso a paso para reproducir los proyectos
- Enfoque en aprendizaje práctico y colaborativo

## Estructura del Proyecto

```
GuardaBotsques/
├── docs/           # Documentación detallada
├── hardware/       # Esquemas y diseños de hardware
├── software/       # Código fuente y programas
├── proyectos/      # Proyectos individuales o en grupo
└── recursos/       # Materiales auxiliares y referencias
```

## Requisitos

- Plataforma de desarrollo Arduino u similar
- Kit básico de robótica (placa de desarrollo, sensores, motores)
- Entorno de programación (Arduino IDE, PlatformIO, etc.)
- Conexión a internet para descargar librerías y actualizaciones

## Instalación

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/IES-ALONSO-QUIJANO/GuardaBotsques.git
   ```

2. Navegar al directorio del proyecto:
   ```bash
   cd GuardaBotsques
   ```

3. Seguir las instrucciones específicas en cada subdirectorio según el proyecto que desee implementar.

## Uso

Cada proyecto dentro del repositorio contiene su propio `README.md` con instrucciones detalladas para:
- Montaje del hardware
- Instalación de software
- Configuración inicial
- Ejecución y prueba
- Solución de problemas comunes
